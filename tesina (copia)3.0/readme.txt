ISTRUZIONI uso booksharing

Per eseguire il programma è necessario avere installato le librerie:
    flask
    flask_bootstrap
    sqlite3

per tanto se non si hanno queste librerie(se invece già installate saltare questi passaggi) andare nel terminale e digitare:
    pip3 install flask
    pip3 install flask_bootstrap


da terminale spostarsi nella cartella del progetto
digitare --> python3 server.py
avviare un browser
inserire url -->  127.0.0.1:5000

Buona navigzione in booksharing!
